# Ambience

## Live

[View on Vercel](https://ambience.vercel.app/)

## Purpose

Give everyone access to world class synthesizer sounds with just the type of a url.

## Running dev Typescript version

```bash
yarn dev
```

## Running dart version

```bash
webdev serve --auto=refresh
```

## Goals

1. Learn Web Audio
2. Learn Web MIDI
3. Learn Dart
4. Learn Web RTC
